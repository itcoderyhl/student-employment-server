# student-employment-server
基于SpringBoot3+Vue3前后端分离的毕业生就业去向管理系统，系统包含管理员、教师、学生角色，功能如下： 管理员：首页统计（学院学生统计、就业去向统计）；学院管理；学生管理；教师管理；就业去向（就业去向、就业方式）；就业协议；档案管理；系统留言；公告信息；修改个人资料；修改密码。 教师：学生管理（本学院）；就业去向（就业去向审核、就业方式）；就业协议（审核）；档案管理（档案处理）；公告信息；修改个人资料；修改密码。 用户：注册；登录；就业去向（就业去向、就业方式查看）；就业协议；档案管理；留言管理；公告信息；个人资料、修改密码。
